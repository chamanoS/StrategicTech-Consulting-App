# Strategic Tech Consulting

A Next.js + Tailwind starter scaffold for a founder-led consulting business with a student academy.

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Ready to extend with MSSQL later

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Pages included

- Home
- Consulting
- Academy
- Case Studies
- Founder
- Contact
